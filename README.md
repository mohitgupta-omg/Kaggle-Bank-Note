# Kaggle-Bank-Note

**Context** : 

Data were extracted from images that were taken from genuine and forged banknote-like specimens. The final images have 400Ã—400 pixels. Due to the object lens and distance to the investigated object gray-scale, pictures with a resolution of about 660 dpi were gained. Wavelet Transform tool were used to extract features from images.

**Content** :

The first four columns are features and the last column is label.

 - Variance of Wavelet Transformed Image
 - Skewness of Wavelet Transformed Image
 - Curtosis of Wavelet Transformed Image
 - Entropy of Image
 
**Output Feature / Label :** 1 - Real ; 0 - Fake
